# ✨ Overview

AI Chatbot Assistant is a full-stack conversational AI platform designed to simulate modern real-time AI assistant experiences.

The application enables users to interact with an AI-powered chatbot capable of handling contextual conversations, dynamic responses, and intelligent interaction workflows using Large Language Models (LLMs).

This project demonstrates practical implementation of:

- AI integration
- conversational workflows
- real-time frontend/backend communication
- REST API architecture
- scalable full-stack AI application design

The system combines modern frontend engineering with AI-powered backend services to deliver responsive conversational experiences.

# ⚙️ Installation

## Step 1: Clone Repository

```bash
git clone https://github.com/malamuru/Chatbot.git
cd Chatbot
```

---

# Backend Setup

```bash
cd server
npm install
```

Create `.env`

```env
PORT=5000
GROQ_API_KEY=your_groq_api_key
CLIENT_URL=http://localhost:5173
```

Run backend:

```bash
npm start
```

---

# Frontend Setup

Open new terminal:

```bash
cd client
npm install
```

Create `.env`

```env
VITE_API_URL=http://localhost:5000/api
```

Run frontend:

```bash
npm run dev
```
